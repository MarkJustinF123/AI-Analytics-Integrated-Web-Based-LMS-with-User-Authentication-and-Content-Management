  The AI Analytics Integrated Web-Based Learning Management System (LMS) is a school project designed to combine an LMS platform with AI-powered analytics for enhanced learning and teaching experiences.
This system enables both students and instructors to manage modules, monitor performance, and visualize learning progress through an easy-to-use web interface. It aims to improve engagement and 
provide insights into academic activities using data-driven analytics.

Features:
- Provides separate access for students and instructors.
- Allows instructors to upload, organize, and manage course materials efficiently.
- Delivers reports and visual insights into student learning patterns and academic performance.
- Ensures secure login and registration for system users.
- Offers a user-friendly and accessible design across multiple devices.

  The system consists of several modules, including user authentication, content management, AI analytics, and a dashboard interface, which will present related information regarding the courses, analytics, and
notifications. It is built using HTML, CSS, JavaScript, PHP, and MySQL, running locally through XAMPP. Mobile app integrations, advanced AI features for predictive learning analysis, and a real-time communication
system for students and instructors are planned for future versions.

    It was developed to demonstrate the combination of AI technology with a traditional learning management system, where academic support and performance
tracking can be provided through one web-based environment.
